Lumos

Post-it notes are ideal for brainstorming and note taking, limiting ideas to succinct content bites and allowing for the quick sharing of ideas. However, for users with mobility issues or a lack of space or post-its, it can be difficult to use this brainstorming method.

Lumos is a voice-controlled memory assistant that riffs off the sticky note culture of the D. School. The app allows for note-taking on virtual sticky notes to create location-aware content bits. process

As this project was created under time constraints for a hackathon, thorough documentation was not made. However, my process involved brainstorming needs for augmented reality technology, as well as brainstorming for functionality using voice control and the pointer finger grab motion sensed by the HoloLens. Various interactions were explored for how to start commands, and what the proper way to edit or delete sticky notes would be.

During the development phase, I created the Lumos helper model in Maya as well as the sticky notes.
